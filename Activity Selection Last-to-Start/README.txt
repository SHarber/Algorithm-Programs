To Compile:
 	g++ -o Activity_Selection Activity_Selection_Last_To_Start.cpp

To Run: 
	./Activity_Selection


Program/Algorith Information: 
	Greedy Algorithm to yield the optimal solution of activity selection.
	Input File:  (act.txt)
		First Row:  Number of Activities in a set
		Following Lines:  Activity Nuber, Start time, Finish time (Space delimited)

	Results:  Output to the Terminal.
		First Line: Set #
		Second Line: Number of Activities Selected
		Third Line:  The Activities Selected.
